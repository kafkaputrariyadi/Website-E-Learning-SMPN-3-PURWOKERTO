BAB I: Pengenalan Komputer

Progress: 100%

Pengertian Komputer

Komputer adalah alat elektronik yang dapat menerima data, memproses data, menyimpan data, dan menghasilkan output berupa informasi. Komputer terdiri dari perangkat keras (hardware) dan perangkat lunak (software) yang bekerja bersama-sama.

Komponen Utama Komputer:
- CPU (Central Processing Unit) - Otak komputer
- RAM (Random Access Memory) - Memori sementara
- Hard Disk - Penyimpanan permanen
- Motherboard - Papan sirkuit utama
- Power Supply - Sumber daya listrik

Jenis-jenis Komputer

Berdasarkan Ukuran:
- Supercomputer (Komputer super)
- Mainframe (Komputer besar)
- Minicomputer (Komputer menengah)
- Microcomputer (Komputer mikro)

Berdasarkan Penggunaan:
- Personal Computer (PC)
- Laptop
- Tablet
- Smartphone
- Server

Perangkat Input dan Output

Perangkat Input:
- Keyboard (papan ketik)
- Mouse (tetikus)
- Scanner (pemindai)
- Microphone (mikrofon)
- Webcam (kamera web)

Perangkat Output:
- Monitor (layar)
- Printer (pencetak)
- Speaker (pengeras suara)
- Projector (proyektor)

BAB II: Sistem Operasi

Progress: 80%

Pengertian Sistem Operasi

Sistem Operasi (Operating System) adalah perangkat lunak yang mengelola sumber daya perangkat keras komputer dan menyediakan layanan umum untuk aplikasi perangkat lunak. Sistem operasi bertindak sebagai perantara antara pengguna dan perangkat keras.

Fungsi Sistem Operasi:
- Manajemen memori
- Manajemen proses
- Manajemen file
- Manajemen perangkat
- Keamanan sistem
- Antarmuka pengguna

Jenis-jenis Sistem Operasi

Windows:
- Dikembangkan oleh Microsoft
- User-friendly dengan GUI
- Banyak digunakan di PC
- Contoh: Windows 10, Windows 11

macOS:
- Dikembangkan oleh Apple
- Khusus untuk komputer Mac
- Tampilan elegan dan stabil
- Contoh: macOS Monterey, macOS Ventura

Linux:
- Open source dan gratis
- Banyak distribusi (distro)
- Stabil dan aman
- Contoh: Ubuntu, Fedora, Mint

BAB III: Microsoft Word

Progress: 60%

Pengenalan Microsoft Word

Microsoft Word adalah program pengolah kata yang digunakan untuk membuat, mengedit, dan memformat dokumen teks. Word merupakan bagian dari Microsoft Office Suite dan sangat populer untuk keperluan penulisan dokumen.

Fitur Utama Microsoft Word:
- Pembuatan dan editing teks
- Pemformatan teks dan paragraf
- Penyisipan gambar dan tabel
- Pemeriksaan ejaan dan tata bahasa
- Template dokumen
- Kolaborasi online

Toolbar dan Menu Penting

Home Tab:
- Font (jenis, ukuran, warna huruf)
- Paragraph (alignment, spacing)
- Styles (gaya teks)
- Editing (find, replace)

Insert Tab:
- Pages (cover page, page break)
- Tables (tabel)
- Illustrations (gambar, shapes)
- Links (hyperlink)

Layout Tab:
- Page Setup (margins, orientation)
- Paragraph (indent, spacing)
- Arrange (position, wrap text)

BAB IV: Microsoft Excel

Progress: 40%

Pengenalan Microsoft Excel

Microsoft Excel adalah program spreadsheet yang digunakan untuk mengolah, menganalisis, dan memvisualisasikan data numerik. Excel sangat berguna untuk perhitungan, analisis data, dan pembuatan grafik.

Konsep Dasar Excel:
- Workbook (buku kerja)
- Worksheet (lembar kerja)
- Cell (sel)
- Row (baris) dan Column (kolom)
- Formula (rumus)
- Function (fungsi)

Fungsi-fungsi Penting Excel

Fungsi Matematika:
- SUM (menjumlahkan)
- AVERAGE (rata-rata)
- MAX (nilai tertinggi)
- MIN (nilai terendah)

Fungsi Logika:
- IF (kondisi)
- AND (dan)
- OR (atau)

Fungsi Teks:
- CONCATENATE (menggabungkan teks)
- LEFT (ambil kiri)
- RIGHT (ambil kanan)
- LEN (panjang teks)

BAB V: Internet dan Email

Progress: 20%

Pengenalan Internet

Internet adalah jaringan global yang menghubungkan jutaan komputer di seluruh dunia. Internet memungkinkan pertukaran informasi, komunikasi, dan akses ke berbagai sumber daya digital.

Layanan Internet:
- World Wide Web (WWW)
- Email (surat elektronik)
- Social Media (media sosial)
- Cloud Storage (penyimpanan awan)
- Video Conference (konferensi video)
- Online Learning (pembelajaran online)

Email dan Etika Berinternet

Komponen Email:
- To (kepada)
- Subject (subjek)
- Body (isi pesan)
- Attachment (lampiran)
- CC (Carbon Copy)
- BCC (Blind Carbon Copy)

Etika Berinternet:
- Gunakan bahasa yang sopan
- Jangan sebarkan informasi pribadi
- Hormati privasi orang lain
- Jangan percaya hoax
- Gunakan sumber yang terpercaya
