CHAPTER I: Descriptive Text

Progress: 100%

Understanding Descriptive Text

Descriptive text is a type of text that aims to describe a person, place, thing, or animal in detail and clearly. This text allows readers to imagine as if they see, feel, or experience directly what is being described.

Characteristics of Descriptive Text:
- Describes objects in detail
- Uses sensory details in description
- Aims to make readers feel like experiencing directly
- Uses vivid and imaginative language
- Focuses on one specific object

Structure of Descriptive Text

1. Identification:
- The opening part that introduces the object
- Mentions the name, type, or category of the object
- Provides general description of the object

2. Description:
- Detailed description of the object's parts
- Describes physical and non-physical characteristics
- Uses descriptive adjectives
- Explains quality, size, color, shape, etc.

Language Features

Specific Participant:
- Uses specific objects (person, place, thing)
- Usually uses pronouns he, she, it

Simple Present Tense:
- Uses present tense form
- Verbs to describe permanent characteristics

Adjectives:
- Uses many descriptive adjectives
- Words describing color, shape, size, taste, etc.

Action Verbs:
- Uses verbs that describe states

CHAPTER II: Procedure Text

Progress: 80%

Understanding Procedure Text

Procedure text is a type of text that contains steps or stages that must be done to achieve a certain goal. This text provides guidance or instructions to readers about how to do something systematically.

Characteristics of Procedure Text:
- Contains sequential steps
- Uses imperative sentences
- Arranged systematically and logically
- Aims to provide practical guidance
- Uses numbering or sequence

Structure of Procedure Text

1. Goal:
- Explains what will be achieved
- Mentions the expected final result
- Provides general overview of the activity

2. Materials:
- List of required materials or tools
- Mentions quantity and specifications
- Can be tools, materials, or equipment

3. Steps:
- Sequence of actions that must be done
- Uses numbering or bullets
- Arranged chronologically

CHAPTER III: Narrative Text

Understanding Narrative Text

Narrative text is a type of text that tells about an event or occurrence chronologically. This text aims to entertain readers by presenting interesting stories with clear plot.

Characteristics of Narrative Text:
- Tells about events or occurrences
- Arranged chronologically (time sequence)
- Has clear story plot
- Contains characters, setting, and conflict
- Aims to entertain or give moral message

Understanding Recount Text

Recount text is a type of text that retells past experiences or events in chronological order. This text aims to inform or entertain readers by sharing personal experiences.

Characteristics of Recount Text:
- Retells past experiences
- Uses chronological order
- Uses past tense
- Contains personal perspective
- Aims to inform or entertain

Understanding Announcement Text

Announcement text is a type of text that contains official information or news addressed to public or specific group of people. This text aims to inform about events, activities, or important information.

Characteristics of Announcement Text:
- Contains important information
- Uses clear and concise language
- Has specific target audience
- Includes essential details (what, when, where)
- Uses formal or semi-formal language
